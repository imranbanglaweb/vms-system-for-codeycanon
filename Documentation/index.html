Introduction

Welcome to InayaFleet360 documentation. This powerful Laravel-based solution is designed to streamline your organization's fleet and transport operations with an all-in-one management platform.

InayaFleet360 is a comprehensive fleet and transport automation system that helps you manage vehicles, drivers, requisitions, maintenance, and more - all in one place.

Installation Guide

Step 1: Download and Setup

Download the package and extract it to your web server directory (e.g., public_html or htdocs).

Step 2: Install Dependencies

Open your terminal and navigate to the project directory:

cd /path/to/your/project
composer install
npm install

Step 3: Configure Environment

Copy the example environment file:

cp .env.example .env

Edit the .env file with your database credentials:

APP_NAME="InayaFleet360"
APP_URL=http://yourdomain.com

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=your_database_name
DB_USERNAME=your_database_username
DB_PASSWORD=your_database_password

Step 4: Generate Application Key

php artisan key:generate

Step 5: Run Database Migrations

php artisan migrate --seed

Step 6: Create Storage Link

php artisan storage:link

Step 7: Set File Permissions

chmod -R 775 storage/
chmod -R 775 bootstrap/cache/
Note: The permissions (775) may need to be adjusted based on your server configuration. Some servers require 755.
Success! Your application should now be accessible at http://yourdomain.com

Module Details

Fleet Management

Manage your vehicle fleet with comprehensive features including vehicle registration, type categorization, document management, and real-time availability tracking.

Driver Management

Maintain driver profiles with license information, document tracking, and availability status for efficient driver assignment.

Transport Management

Handle transport requisitions with multi-level approval workflows, create trip sheets, assign vehicles and drivers, and track all transport activities.

Task Management

Schedule and track maintenance for vehicles and equipment, manage maintenance categories, and coordinate with vendors.
